# Spirit Island Toolbox

Welcome to Spirit Island Toolbox. This is a webapp created to replace the invader board for Spirit Island, to reduce the amount of setup needed when playing the board game. 

# Features

##### Turn Tracking
Turns are automatically tracked and advanced with a single button press. The indicator pane updates dynamically depending on what is coming next, including invader actions and adversary-specific actions.
